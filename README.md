## Overview
This project is a production-grade backend service for a real-time collaborative workspace platform.  
It supports authentication, project & workspace management, real-time collaboration, and asynchronous job processing.

The system is designed with scalability, security, and clean architecture principles.

---

## Tech Stack
- **Backend Framework:** NestJS (TypeScript)
- **Database:** PostgreSQL (Prisma ORM)
- **Authentication:** JWT + Passport
- **Real-Time Communication:** WebSockets (Socket.IO)
- **Async Jobs:** BullMQ + Redis
- **ORM:** Prisma
- **API Style:** REST + WebSocket
- **Deployment Ready:** Docker / Cloud compatible
